# 02 — Visual Style Guide for AI Workflows Website

## Brand feeling

AI Workflows should feel:

- Premium
- Intelligent
- Trustworthy
- Fast-moving
- Practical for Indian businesses
- Not too corporate and not too childish

## Visual keywords

Use these keywords while designing:

- AI automation
- Modern business software
- Local business growth
- Premium dashboards
- Smart workflows
- Reliable technology partner
- Human + AI team

## Visual direction

### Primary style

Dark AI SaaS theme with purple-blue gradients.

### Secondary style

Clean white/light sections can be used only inside cards, screenshots, dashboards, or demo previews. Keep the main background dark.

## Main design principles

### 1. Clarity before decoration

Every section must answer one business question:

- What do you do?
- Who is it for?
- How does it help?
- Can I see proof?
- How much does it cost?
- How do I contact you?

### 2. Premium spacing

Use more whitespace. Do not crowd everything.

Recommended section padding:

```css
section {
  padding-top: 96px;
  padding-bottom: 96px;
}
```

Mobile:

```css
section {
  padding-top: 64px;
  padding-bottom: 64px;
}
```

### 3. One strong gradient style

Do not use too many random colors.

Use purple, blue, cyan, and warm gold only.

### 4. Consistent icon style

Use one icon library only.

Recommended:

- `lucide-react` for UI icons
- Emoji can be used rarely in hero cards, but avoid overuse

### 5. Use screenshots/mockups

Your product work should be visible through:

- Website screenshot frames
- Dashboard cards
- Mobile chat mockups
- Workflow diagrams
- Before/after panels

## Recommended page mood

### Hero

Bold, energetic, high contrast.

### Services

Structured and easy to compare.

### Industries

Practical and business-focused.

### Projects

Visual proof-heavy.

### Process

Step-by-step, confidence-building.

### Pricing

Simple and transparent.

### Contact

Friendly and direct.

## Premium UI elements to add

- Sticky glass navbar
- Gradient headline text
- Animated background orbs
- Floating dashboard cards
- Product demo cards
- Scroll reveal sections
- Metric counters
- Timeline process
- Testimonial/proof cards
- Pricing cards with highlighted recommended plan
- Contact CTA panel

## Avoid

- Too many emojis
- Too many different fonts
- Too many glowing effects
- Tiny text
- Low contrast grey text
- Random animations
- Overloaded navbar
- Too much text in one section
- Generic stock images without product relevance
