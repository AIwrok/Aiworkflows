# Product Demo Pages Plan

## Purpose

Create separate landing/demo pages so each client sees only the solution relevant to their business.

This is important because a restaurant owner, cyber café owner, and medical distributor have different problems. One homepage cannot explain all of them deeply.

## Design guidance

- Mobile-first layout
- Big heading
- Short paragraphs
- Use workflow arrows
- Use 3–4 screenshot cards
- Keep business value above technical details
- Use Hindi-friendly simple English
