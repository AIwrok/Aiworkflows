# 01 — Design Audit of Current AI Workflows Homepage

## Current strong points

From the current screenshot:

- Dark premium background already gives a modern AI company feeling.
- Purple gradient CTA button looks attractive.
- Hero headline is bold and visible.
- Navigation has important sections: Services, Industries, Live Links, Projects, Proof, Process, Team, Pricing, Contact.
- The website already positions AI Workflows as a business automation company, not only a website provider.

## Main improvement areas

### 1. Logo and header polish

Current issue:

- Logo area feels slightly oversized and not fully premium.
- Header spacing can be more balanced.
- Navigation items are many, so the header may feel crowded.

Improve with:

- Use fixed logo height around `48px`.
- Use smaller nav font size: `14px` or `15px`.
- Add active/hover states with gradient underline.
- Make header glass/sticky with blur.
- Add mobile hamburger menu with smooth slide panel.

### 2. Hero section hierarchy

Current issue:

- Hero headline is strong, but supporting text and CTA area need more conversion power.
- Right-side cards look good, but can be more dynamic and structured.

Improve with:

- Add clear subheadline:
  `We build websites, dashboards, WhatsApp automation, and AI workflows that help Indian businesses save time and get more customers.`
- Add two CTA buttons:
  - Primary: `Book Free Demo`
  - Secondary: `View Live Projects`
- Add trust line:
  `Built for restaurants, cyber cafes, distributors, clinics, schools, and local SMBs.`
- Add animated business cards on the right.

### 3. Card design

Current issue:

- Cards are useful, but the website needs a consistent card system.

Improve with:

- All cards should use same radius, border, shadow, hover state.
- Add icons in consistent style.
- Add subtle gradient border on hover.
- Use "problem → solution → result" format.

### 4. Section separation

Current issue:

- Dark sections can merge into each other if backgrounds are too similar.

Improve with:

- Add section labels.
- Add top/bottom spacing.
- Use alternating background gradients.
- Use subtle grid/noise pattern.
- Use glowing divider lines.

### 5. Proof and trust

Current issue:

- Client may ask: "Have you done this before?"
- Current site needs more proof visuals.

Improve with:

- Add demo screenshots.
- Add "Live product demos" cards.
- Add result-based proof:
  - `Save 4+ hours/day`
  - `Launch website in 7 days`
  - `Automate repeat customer queries`
  - `Track orders and delivery`

### 6. Animation balance

Current issue:

- Professional dynamic feel requires motion, but too much animation will look cheap.

Improve with:

- Smooth hero fade-up.
- Scroll reveal cards.
- Hover lift on cards.
- Animated gradient orbs.
- Slow moving background grid.
- Counter animation for metrics.
- Avoid fast bouncing or distracting effects.

## Design direction

Use this style direction:

> Dark premium AI SaaS + Indian business trust + clean conversion-focused agency website.

## Final output expectation

After redesign, a new visitor should understand in 5 seconds:

1. What AI Workflows does.
2. Who it serves.
3. Why it is better than a normal website agency.
4. Which live demos/projects can be checked.
5. How to book a demo.
