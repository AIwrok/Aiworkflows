Each proof card should include:


## Case study layout

Create case study pages later using this structure:

```txt
/case-studies/restaurant-automation
/case-studies/cyber-cafe-ai-form-filler
/case-studies/medical-distributor-software
```

Each case study should include:

1. Business problem
2. Existing manual workflow
3. AI Workflows solution
4. Screenshots
5. Step-by-step flow
6. Business result
7. What can be customized
8. CTA

## Screenshot checklist

Before adding screenshots, confirm:

- [ ] No private customer data visible
- [ ] No real phone numbers visible
- [ ] No admin password/API key visible
- [ ] Screenshot is clear on mobile
- [ ] Screenshot has proper spacing
- [ ] UI looks premium
- [ ] Image is compressed
- [ ] Alt text is added
