# AI Workflows Website Improvement Pack

This folder contains practical `.md` files to improve the AI Workflows website and make it stronger for client meetings, demos, and lead generation.

## Goal

Convert `https://ai-workflows.cloud/` from a good company website into a stronger sales website that can:

- Build trust quickly
- Show clear product demos
- Explain business value in simple words
- Capture leads from business owners
- Help the team work phase-by-phase
- Support LinkedIn/social media promotion

## Recommended execution order

1. Start with `01-master-improvement-todo.md`
2. Build product landing pages using `02-product-demo-pages-plan.md`
3. Improve pricing with `03-pricing-packages-copy.md`
4. Add proof/case-study sections using `04-client-proof-and-case-study-sections.md`
5. Improve lead capture using `05-lead-generation-and-cta-improvements.md`
6. Start content marketing using `06-seo-and-linkedin-content-plan.md`
7. Give developers the prompts from `07-codex-claude-implementation-prompts.md`
