# Lead Generation and CTA Improvements

## Purpose

Improve the website so more visitors become leads.

The website should not only explain services. It should guide business owners toward a demo or WhatsApp conversation.

---

## Primary CTA

Use one main CTA across the website:

```txt
Book Free Business Automation Demo
```

Why this works:

- Sounds valuable
- Not limited to website
- Explains that the meeting is about business improvement
- Good for restaurants, cyber cafֳ©s, distributors, shops, and service businesses

## Mobile sticky CTA

On mobile, add bottom sticky bar:

```txt
[WhatsApp] [Book Demo]
```

Rules:

- Should not cover important content
- Should be small but visible
- Should not feel spammy
- Should load fast
